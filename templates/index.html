Welcome to SQLi Labs

Brought to you by Krithika

SQL injection is an attack in which the hacker injects malicious SQL code into input fields of an application. By doing this, they get access to sensitive information stored in the database, and they can modify it as well. This happens when the application hasn't used parameterized queries, which makes it possible for the hacker to inject malicious SQL statements.

I have built this application which is deliberately vulnerable, to help you simulate and understand how SQLi attacks work.

Now, you will face three challenges at easy, medium and hard levels, with clues to help. Ready to perform some SQL injections?
